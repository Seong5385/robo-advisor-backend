# XAI 기반 주식 로보어드바이저 시스템 요구사항 명세서

------------------------------------------------------------------------

## 1. 프로젝트 개요

### 1.1 프로젝트명

XAI 기반 주식 로보어드바이저 시스템

### 1.2 개발 목적

-   사용자 투자 성향 분석 기반 맞춤형 포트폴리오 추천
-   AI 주가 예측 모델 제공
-   예측 결과에 대한 설명(XAI) 기능 제공
-   뉴스 감성 분석 기반 투자 의사결정 지원

### 1.3 개발 환경

  구분       기술
  ---------- -----------------
  Backend    Spring Boot
  ORM        Spring Data JPA
  Database   PostgreSQL
  Frontend   React, Next.js
  배포       Docker
  형상관리   Git

------------------------------------------------------------------------

## 2. 기능 요구사항

### 2.1 사용자 관리 기능

#### 2.1.1 회원가입

-   사용자 정보 입력
-   비밀번호 암호화 저장
-   이메일/아이디 중복 검사
-   회원가입 후 투자 성향 설문 진행

#### 2.1.2 로그인

-   JWT 기반 인증
-   Access / Refresh Token 발급
-   인증 실패 처리

#### 2.1.3 회원 관리

-   회원 정보 수정
-   비밀번호 변경
-   회원 탈퇴

------------------------------------------------------------------------

### 2.2 투자 성향 설문 기능

#### 2.2.1 설문 응답

-   점수 기반 성향 분류 (공격형 / 중립형 / 안정형)
-   설문 재진행 가능

#### 2.2.2 성향 산출

-   점수 계산 로직
-   기준 점수에 따른 Risk Type 결정
-   사용자 테이블 업데이트

#### 2.2.3 설문 이력 관리

-   설문 결과 히스토리 저장
-   설문 변경 시 기존 기록 유지

------------------------------------------------------------------------

### 2.3 대시보드 기능

#### 2.3.1 사용자 정보 표시

-   현재 투자 성향
-   총 자산
-   수익률

#### 2.3.2 보유 종목 관리

-   보유 종목 목록 조회
-   평균 매입가
-   현재가 대비 수익률 계산

#### 2.3.3 실시간 주가 조회

-   외부 API 연동
-   캐시 전략 적용

------------------------------------------------------------------------

### 2.4 AI 예측 모델 기능

#### 2.4.1 주가 예측

-   종목 선택 후 미래 가격 예측
-   예측 그래프 제공
-   상승/하락 확률 제공

#### 2.4.2 모델 관리

-   모델 버전 관리
-   재학습 가능 구조 설계
-   예측 결과 DB 저장

------------------------------------------------------------------------

### 2.5 XAI 설명 기능

#### 2.5.1 예측 설명 제공

-   예측 결과에 영향을 준 주요 지표 표시
-   뉴스 감성 점수 반영 여부 설명
-   기술적 지표 기여도 표시

#### 2.5.2 사용자 친화적 설명 생성

-   자연어 기반 예측 설명 제공

------------------------------------------------------------------------

### 2.6 뉴스 감성 분석 기능

#### 2.6.1 뉴스 수집

-   종목 관련 뉴스 크롤링
-   뉴스 데이터 저장

#### 2.6.2 감성 분석

-   긍정/부정 점수 계산
-   종목별 감성 점수 시각화

------------------------------------------------------------------------

### 2.7 관리자 기능

-   전체 사용자 수 조회
-   투자 성향 분포 통계
-   예측 모델 정확도 통계
-   시스템 로그 조회

## 3. 비기능 요구사항

### 3.1 보안

-   비밀번호 암호화

-   CORS 설정
-   SQL Injection 방지
-   HTTPS 적용

### 3.2 성능

-   예외 처리 통합
