# 🚀 XAI 기반 주식 로보어드바이저 시스템

> **Explainable AI 기반 투자 성향 분석 및 주가 예측 로보어드바이저
> 플랫폼**

------------------------------------------------------------------------

## 📌 1. 프로젝트 개요

### 🎯 프로젝트 목적

-   사용자 **투자 성향 분석** 기반 맞춤형 포트폴리오 추천
-   AI 기반 **주가 예측 모델 제공**
-   예측 결과에 대한 **설명 가능한 AI(XAI) 기능 구현**
-   뉴스 감성 분석 기반 투자 의사결정 지원

------------------------------------------------------------------------

## 🛠 기술 스택

  구분              기술
  ----------------- -----------------
  Backend           Spring Boot
  ORM               Spring Data JPA
  Database          PostgreSQL
  Frontend          React, Next.js
  Deployment        Docker
  Version Control   Git

------------------------------------------------------------------------

# 🧩 2. 주요 기능

## 👤 사용자 관리

### 🔐 회원가입

-   사용자 정보 입력
-   비밀번호 암호화 저장
-   이메일 / 아이디 중복 검사
-   가입 후 투자 성향 설문 자동 진행

### 🔑 로그인

-   JWT 기반 인증
-   Access / Refresh Token 발급
-   인증 실패 처리

### ⚙ 회원 관리

-   회원 정보 수정
-   비밀번호 변경
-   회원 탈퇴

------------------------------------------------------------------------

## 📊 투자 성향 설문

-   점수 기반 성향 분류 → **공격형 / 중립형 / 안정형**
-   설문 재진행 가능
-   설문 결과 히스토리 저장
-   Risk Type 자동 산출 및 사용자 정보 업데이트

------------------------------------------------------------------------

## 📈 대시보드

### 📌 사용자 정보

-   현재 투자 성향
-   총 자산
-   누적 수익률

### 📌 보유 종목 관리

-   보유 종목 목록 조회
-   평균 매입가 계산
-   실시간 수익률 계산

### 📌 실시간 주가 조회

-   외부 API 연동
-   캐시 전략 적용

------------------------------------------------------------------------

## 🤖 AI 예측 모델

-   종목 선택 후 미래 가격 예측
-   예측 결과 그래프 제공
-   상승 / 하락 확률 제공
-   모델 버전 관리
-   예측 결과 DB 저장

------------------------------------------------------------------------

## 🧠 XAI 설명 기능

-   예측 결과에 영향을 준 주요 지표 시각화
-   뉴스 감성 점수 반영 여부 설명
-   기술적 지표 기여도 표시
-   자연어 기반 예측 설명 생성

> 예:\
> "최근 거래량 증가와 긍정 뉴스 비율 상승으로 인해 상승 확률이 높게
> 예측되었습니다."

------------------------------------------------------------------------

## 📰 뉴스 감성 분석

-   종목 관련 뉴스 수집
-   뉴스 데이터 저장
-   긍정/부정 감성 점수 계산
-   종목별 감성 점수 시각화

------------------------------------------------------------------------

## 🛠 관리자 기능

-   전체 사용자 수 조회
-   투자 성향 분포 통계
